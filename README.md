# Duckie – Autonomous Water Monitoring Vehicle

Duckie is a small robotic water vehicle designed to collect water quality data from lakes and reservoirs.

This repository contains:

- `firmware/` – ESP32 microcontroller code
- `backend/` – Simple API + data viewer
- `infrastructure/` – Notes/scripts for connecting the device to AWS IoT and storing data in S3

> This project started as a collaboration between mechanical, electrical, and software workstreams; this repo focuses on the software side – firmware and cloud backend.
